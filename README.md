# 起身吧 (Arise)

macOS 菜单栏久坐提醒应用，基于 Electron 构建。

## 功能

- 菜单栏实时显示倒计时
- 到时间后发送系统通知，提醒你站起来活动
- 支持设置提醒间隔：15 / 30 / 45 / 60 / 90 / 120 分钟
- 支持暂停 / 恢复 / 立即重置计时器

## 截图

菜单栏显示倒计时，点击查看菜单：

```
距下次提醒: 58:30
──────────────
⏸ 暂停提醒
🔄 立即重置
──────────────
提醒间隔 ▸  15 分钟 / 30 分钟 / ...
──────────────
退出
```

## 安装与运行

```bash
# 安装依赖
yarn install

# 开发运行
yarn start

# 打包
yarn build
```

打包产物在 `dist/` 目录下，解压 zip 或直接使用 `.app` 文件即可。

## 技术栈

- [Electron](https://www.electronjs.org/)
- macOS Tray API + Notification API
